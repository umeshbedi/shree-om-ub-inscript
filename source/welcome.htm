Start Using SHREE OM (UB) INSCRIPT

Thank you for installing SHREE OM (UB) INSCRIPT, an InScript-style Hindi keyboard designed for use with the ShreeLipi (legacy) font.

To begin typing:

  • Select SHREE OM (UB) INSCRIPT from the Keyman tray icon
  • Open any text editor (e.g., Notepad)
  • Set the font to ShreeLipi
  • Type using the standard InScript layout

Keyboard Layout

Normal Key and SHIFT + Key




RIGHT ALT + Key and RIGHT ALT + SHIFT + Key

Notes

This keyboard is intended for legacy document editing and printing workflows. For modern applications, Unicode Hindi keyboards and fonts are recommended.
