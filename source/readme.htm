SHREE OM (UB) INSCRIPT

SHREE OM (UB) INSCRIPT is an InScript-style Hindi keyboard designed for use with the ShreeLipi (legacy) font. This keyboard is intended for users who work with existing ShreeLipi-based documents and prefer the standard InScript layout.

Key Features

  • InScript-style Hindi keyboard layout
  • Compatible with ShreeLipi (legacy) font encoding
  • Supports common conjuncts and matras
  • Built using Keyman
  • Free and open-source (GPL-3.0)

Usage Notes

To use this keyboard correctly, ensure that the ShreeLipi font is installed on your system. Select the keyboard from the Keyman tray icon and use it in any text editor.

This keyboard does not distribute ShreeLipi fonts and does not claim ownership of the ShreeLipi encoding. Users must obtain and use ShreeLipi fonts legally.
